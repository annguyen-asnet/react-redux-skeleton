# React-Redux & Immutable & Webpack - Skeleton
This is the basic skeleton that use to create a new react app with redux and webpack.

## Features:
*	Compilation with webpack
*	React and jsx
* React-redux
*	Stylesheets CSS, SASS
*	Unit testing with Mocha
*	Immutable
*	Es6-promise
*	React-router

## Getting Started
```
	npm install
	npm start
```
